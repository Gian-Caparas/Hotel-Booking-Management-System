PROJECT OVERVIEW

The Hotel Booking Management System is a Java Desktop Application designed to handle room
reservations, customer management, and administrative operations for a hotel with 35 rooms across three
floors. The system follows the Model-View-Controller (MVC) architecture and demonstrates all four pillars
of Object-Oriented Programming.
